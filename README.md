<h1 align="center">📈 Sigmoid Function & Logistic Regression Basics</h1>

<p align="center">
This project demonstrates the implementation and visualization of the <b>Sigmoid Function</b>, 
a fundamental component of <b>Logistic Regression</b> used in binary classification problems.
It shows how linear model outputs can be transformed into probabilities between 0 and 1.
</p>

---

<h2>🚀 Project Overview</h2>

<ul>
<li>Implements the sigmoid function from scratch</li>
<li>Demonstrates how logistic regression produces probabilities</li>
<li>Visualizes the sigmoid curve</li>
<li>Explains how classification thresholds work</li>
<li>Builds intuition for binary classification models</li>
</ul>
